Readwise Reader 是一款集合了多种信息渠道和源头的阅读器，包含但不限于 Twitter、YouTube 视频、PDF、EPUB 电子书、网页文章等。

本项目核心是解决在浏览 Twitter 上的文章或帖子时，如何快速将其保存到阅读器中，以便进行深入学习或阅读。

# 1. 官方方案：
   - 在评论区艾特官方账号 @Readwise，并写下“save thread”（保存线程）的评论。
   - 这种方式操作略显繁琐，需要你在想要保存的帖子下手动评论。
   - 完成后，它会提示内容已保存。如果是第一次使用，按照指引完成初始化配置即可。

# 2. 本项目方案（篡改猴脚本）：
   - 本项目提供了一个更简单的方案，核心是一个篡改猴（Tampermonkey）脚本。
   - 安装后，脚本会在网页版帖子的底部增加一个新按钮。该按钮位于点赞、转发、评论、分享那一排的最右端，图标类似一个复杂的笔记本。
   - 首次使用配置：同时按住“鼠标左键”和键盘的“Alt 键”，网页会弹出一个输入框。你需要填入 Readwise 阅读器的 TOKEN（可以自行搜索获取）。
   - 填入 TOKEN 后，再次点击帖子底部的按钮，即可自动保存内容。

# 3.总结

相较于官方提供的评论方式，这种方式更加便捷，尤其针对长信息流的帖子（即博主通过不断追加评论形成的“楼层式”帖子）。

使用该脚本，你只需要点击第一个帖子的按钮，后续对应的所有内容都会被自动保存到阅读器中，非常方便。

以上就是本项目的基本介绍，希望能对大家有所帮助！
